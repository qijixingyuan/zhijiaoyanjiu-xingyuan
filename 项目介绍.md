# 中国高职院校政策研究平台 — 项目介绍

## 项目背景

面向高职教育政策研究人员，搭建 1540 所中国高职（专科）院校的政策研究工具。数据来源于教育部《全国普通高等学校名单》（截至 2026-06-17）。

## 技术栈

| 层 | 方案 |
|----|------|
| 前端框架 | Next.js 14 App Router + TypeScript |
| 样式 | Tailwind CSS 3 |
| 地图 | ECharts 5 + echarts-for-react + DataV GeoJSON |
| 数据库 | SQLite (Prisma ORM)，可迁移 PostgreSQL |
| 图表 | ECharts 5 |
