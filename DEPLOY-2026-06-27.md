5. **Nginx 隔离**: 新增 location/代理，不修改 luoyu 的配置

## 部署命令

```bash
# 1. 连接服务器
ssh -i "D:\AI Projects\claude cunjifen\01系统基础指南\luoyu.pem" root@47.107.31.231

# 2. 拉取最新代码
cd /www/wwwroot/zhijiao
git pull

# 3. 构建（端口 3001）
PORT=3001 npm run build

# 4. 启动 PM2
pm2 start npm --name zhijiao -- start -- --port 3001
pm2 save

# 5. 验证
curl http://127.0.0.1:3001/

# 6. 查看日志
pm2 logs zhijiao
```
