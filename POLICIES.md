## 三、技术问题

### 问题 1: 教育部 SPA 页面不稳定

- **现象**: `https://www.moe.gov.cn/s78/A07/` 有时 30 秒超时
- **原因**: 教育部网站是 Vue.js SPA，有反爬机制
- **影响**: 全国级政策（最重要的来源）不稳定
- **尝试过的方案**:
  - `networkidle` waitUntil → 仍然偶发超时
  - 添加 `waitForSelector` → 选择器不出现时回退通用提取
- **待尝试**: 使用 `stealth` 模式、增加重试、使用其他 MOE 政策页面

### 问题 2: 关键词匹配不精准

- **现象**: 大量通用教育通知被判定为"职教政策"
- **原因**: `MUST_MATCH` 正则包含"招生""院校"等过于宽泛的关键词
- **影响**: 类型分类失衡 — 48.4% 被归类为 A-治理体系（默认类型）
- **待改进**: 细分关键词、改进 `classifyPolicy` 算法

### 问题 3: 类型分类过于依赖关键词匹配

- **现象**: `classifyPolicy()` 只用 `title.includes(keyword)`，逻辑简单
- **影响**:
  - B-产教融合: 2 条（实际应该更多，政策标题不一定包含"产教融合"字面）
  - E/I/J/L: 0 条（关键词覆盖不足）
  - A-治理体系作为 default，吞噬了大量应归入其他类型的政策
- **待尝试**: LLM 辅助分类、扩展关键词列表、增加 summary 分析

### 问题 4: HTML 日期提取不完整

- **现象**: 71 条政策无 URL、41 条无部门信息
- **原因**: 通用提取器只能从 `<span>`/`<em>` 等简单标签提取日期，很多省份使用特殊 HTML 结构
- **待改进**: 为每个源写专用日期提取器，从 URL 路径解析日期

### 问题 5: 日期提取不准确 → 政策显示爬取日期而非发布日期（2026-06-27 发现）

- **现象**: 湖北省"推动现代职业教育高质量发展的实施意见"显示日期为 2026-06-26（爬取日期），实为 2022-04-15
- **根因**: `extractDateFromUrl()` 缺少湖北 `tYYYYMMDD_` 和河南 `YYYY/MM-DD` 两种 URL 模式
- **影响范围**: 共 8 条政策日期错误（湖北 5、全国 2、其他 1），7 条广东政策因 URL 无日期仍使用 `new Date()`
- **已修复**:
  - ✅ 新增 URL 模式: `/tYYYYMMDD_`（湖北）、`YYYY/MM-DD`（河南）、`/YYYYMM/YYYYMMDD_`（福建）
  - ✅ 数据库 8 条错误日期已修正
  - ⚠️ 广东 `/content/post_XXX.html` 格式仍无法提取（URL 中无日期，需 HTML 日期提取）
- **教训**: 凡是依赖 `new Date()` 作为日期回退值的地方，数据都会泄露爬取时间

### 问题 6: 服务器上 GitHub 无法 HTTPS 访问

- **现象**: 服务器 `git pull` 超时
- **已修复**: 切换为 SSH (git@github.com)
- **状态**: ✅ 已解决

### 问题 7: 爬虫仅在本地运行

- **现象**: 服务器上的数据是手动上传的 `dev.db`
- **影响**: 服务器数据不更新，需本地运行爬虫后重新上传
- **待尝试**: 在服务器上运行爬虫脚本（安装 Playwright 依赖）

## 四、结构/逻辑问题

### 问题 8: 省份分布极度不均

- 21 个省份只有 1-2 条政策
- 但 31 省在 UI 上显示为"全覆盖"
- 用户可能误以为数据完整

### 问题 9: 没有增量爬取机制（已部分修复 2026-06-29）

- ✅ 翻页已实现：从第1页自动发现 `index_N.html` / `?pageNum=N` 链接
- ✅ 广东翻5页（18→74）、江苏翻4页（5→12）、山东翻2页
- ✅ 提前终止：某页无新数据时自动停止
- ⚠️ 湖南翻页链接已发现但 `document.write` 导致提取失败
- ⚠️ 仍无"上次爬取位置"记忆，每次从头爬

### 问题 10: 没有政策去重的 content-based 方案

- 去重仅依赖 URL，同一政策在不同 URL 下会重复
- 没有基于标题相似度的去重

### 问题 10: 种子数据与爬虫数据的日期范围不一致

- 日期过滤设为 2025-01-01
- 但数据库中存在 2013 年的数据（来自早期无过滤的爬取）
- 需统一清理或调整过滤逻辑

## 五、文字/内容问题

### 问题 11: 政策缺少 summary（摘要）

- 所有政策的 `summary` 字段为空
- 政策列表只显示标题 + 类型标签，用户无法快速了解政策内容
- 需要: 爬取政策正文第一段、或用 LLM 生成摘要

### 问题 12: department 字段不准确

- 41 条政策无部门信息
- 有的被标记为 `sourceOrg` 而非实际发文部门
- "教育部-职成司" vs "教育部职业教育与成人教育司" 名称不一致

### 问题 13: 部分政策标题含 HTML 实体或乱码

- 从 `document.write(str.substr(...))` 页面提取的标题可能有截断
- 部分标题末尾含多余空格或不可见字符
