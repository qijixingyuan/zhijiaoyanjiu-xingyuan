## 一、政策爬虫 v3

### 问题
- v2 只有 6 个源，多数失效（404/SPA/SSL错误）
- 79 条政策，覆盖不全

### 修复
- 重写为通用提取器：Playwright `networkidle` + 通用 link 提取 + Node 侧关键词过滤
- 修复 5 个源的 URL：
  - 教育部：`/jyb_xxgk/xxgk_zjjy/zcfg/` → `/s78/A07/`（职成司）
  - 湖南：`document.write` 问题 → `networkidle` 解决
  - 广东：`gsgg/` → `jyzcfg/`（政策法规专栏）
  - 江苏：`col57890/`（404）→ `col58320/`（通知公告）
  - 浙江：`col1543966/`（404）→ `col1532983/`（职业教育专栏）
- 新增日期范围过滤：仅 2025-01-01 至今

### 扩展（用户提供 URL）

|------|-----|:--:|
| 山东 | `edu.shandong.gov.cn/col/col11990/` | ✅ 4条 |

### 修复的 bug
- **山东**: `networkidle` 在 HTTP 站点超时 → 改用 `domcontentloaded`，增加 `waitUntil` 配置
- **湖北**: URL 为 `.shtml` 且路径含 `/zfxxgk/`，提取器未识别 → 补全模式

### 结果
- 79 → 122 条政策（+54%），覆盖全国 31 省
- 爬虫源 11 个（国家级 1 + 省级 10）

## 二、院校详情 UI 重构

### 需求
- 筛选后列表出现在**右侧网页框**，详情变成**弹出窗口覆盖**

### 改动

| 文件 | 操作 | 说明 |
|------|:--:|------|
| `CollegeDetailContent.tsx` | **新建** | 提取 170 行重复代码为共享组件（4-tab详情） |

### 隔离规则
1. 端口隔离：3001 ≠ 3000
2. 进程隔离：PM2 名 `zhijiao` ≠ `luoyu-backend`
3. 目录隔离：`/www/wwwroot/zhijiao/` ≠ `/www/wwwroot/luoyu/`
4. 数据库隔离：SQLite 文件 ≠ MySQL
5. Nginx 隔离：不修改现有配置

### 部署三原则（用户要求）
1. 先提交到 git，push 后再 pull 到服务器
2. 服务器热修复后，必须同步回本地并提交 git
3. 直接传文件前先 `git status` 确认无冲突
