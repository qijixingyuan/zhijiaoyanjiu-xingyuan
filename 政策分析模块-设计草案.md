# 政策分析模块 — 设计草案

## 5 个学术维度

### 1. 政策演进趋势
- 按年×类型堆叠面积图
- 显示各类型政策的年度分布变化

### 2. 省份政策活跃度
- 条形图排名 + 地图热力
- 省份 × 类型交叉统计

### 3. 政策主题共现
- 多标签 → 共现矩阵 → 网络图
- 需要先将单选 type 改为多标签

### 4. 发文主体分析
- department 字段分组统计
- 饼图 + 联合发文检测

### 5. 政策密集度时序
- 按月份 × 类型日历热力图
- 识别政策发布季节规律

## 优先实施

1. 演进趋势（数据最全）
2. 省份对比（直接可用）
3. 共现分析（需改数据模型）
